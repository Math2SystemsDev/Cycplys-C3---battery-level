# Cycplus C3 Battery level

Uma breve descrição sobre o que esse projeto faz e para quem ele serve.


## principal
- conectar seu dispositivo cycplus c3 via bluetooth na aplicação web;
- mostra o nível de bateria atual em porcentagem baseado em dados enviado do seu próprio dispositivo conectado.

## Observações

- Para testar no smartphone android é necessário ativar "Experimental Web Platform features" no Google Chrome Android e em seguida reiniciar o aplicativo.

- Lembre-se de ativar o bluetooth e parear somente dentro da aplicação.
